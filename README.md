## Setup

- Copy members.csv to `public/members.csv`
- Edit [custom styles & tier name regex](./src/CreditRoll.tsx#L30-L44)

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```
